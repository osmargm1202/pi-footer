# Client Mail Patterns

## Send document

```text
Saludos,

Adjunto al correo [DOCUMENTO] para fines de revisión.

Favor verificar y dejarme saber cualquier observación o ajuste requerido.

Gracias.
```

## Request information

```text
Saludos,

Para continuar con [PROYECTO], favor remitir la siguiente información:

- [INFORMACIÓN 1]
- [INFORMACIÓN 2]

Quedo atento.

Gracias.
```

## Follow-up

```text
Saludos,

Doy seguimiento a [TEMA] para confirmar si tienen alguna observación o si podemos proceder con el próximo paso.

Quedo atento a sus comentarios.

Gracias.
```

## Respond to observations

```text
Saludos,

Recibidas las observaciones. Debajo dejo el resumen de los ajustes considerados:

- [AJUSTE 1]
- [AJUSTE 2]

Favor verificar y confirmar si queda conforme.

Gracias.
```
