# Proposal Structure

## Standard sections

1. Objetivo
2. Alcance contemplado
3. Entregables
4. Exclusiones
5. Supuestos y condiciones
6. Tiempo de entrega
7. Inversión o base económica
8. Próximos pasos

## Preferred wording

- `El presente documento tiene como finalidad...`
- `El alcance contempla...`
- `Se consideran los siguientes entregables...`
- `No se incluyen...`
- `Para continuar, se requiere...`
- `Quedo atento a cualquier observación.`

## Short proposal skeleton

```text
Saludos,

Debajo dejo la propuesta correspondiente a [PROYECTO].

## Objetivo
[OBJETIVO]

## Alcance contemplado
- [ALCANCE 1]
- [ALCANCE 2]

## Entregables
- [DOCUMENTO]
- [DOCUMENTO]

## Exclusiones
- [EXCLUSIÓN]

## Próximos pasos
Favor revisar y confirmar si podemos proceder con lo indicado.

Gracias.
```
