# Technical Document Structure

## Memoria técnica

1. Objetivo
2. Alcance
3. Criterios considerados
4. Descripción técnica
5. Observaciones
6. Conclusiones o próximos pasos

## Reporte técnico

1. Resumen
2. Situación observada
3. Acciones realizadas
4. Resultado
5. Pendientes o recomendaciones

## Procedimiento

1. Propósito
2. Requisitos previos
3. Pasos
4. Validación
5. Observaciones

## Preferred wording

- `El presente documento tiene como finalidad...`
- `Se consideran los siguientes criterios...`
- `Durante la revisión se observó...`
- `Se recomienda...`
- `Para continuar, se requiere...`

## Technical tone

Use a direct explanation of what was reviewed, what was found, what was adjusted, and what remains pending. Do not overstate certainty when the provided information is incomplete.
